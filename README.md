# YouTube Time Tracker

A Chrome extension that tracks your YouTube watch time and helps you stay focused.

## Features

**Time Tracking**
- Track daily watch time and total time on YouTube
- Weekly watch time summary
- Live timer displayed in the YouTube header

**Time Limits**
- Set a daily watch limit
- Get blocked when you exceed your limit
- "5 more minutes" button for when you need just a bit more

**Focus Mode**
- Shorts are always blocked (redirects to home)
- Optional: Block side recommendations on video pages
- Optional: Block home feed to avoid endless browsing

## Installation

1. Clone or download this repository
2. Open `chrome://extensions` in Chrome
3. Enable "Developer mode" (top right)
4. Click "Load unpacked" and select this folder

## Usage

Click the extension icon to:
- View your watch stats
- Adjust your daily limit
- Toggle focus features

The extension automatically tracks time when you're watching videos and displays your stats in the YouTube header.
